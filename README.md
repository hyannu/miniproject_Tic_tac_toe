# 🎮 Tic Tac Toe Game

A modern Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. 
This project demonstrates core frontend development concepts, 
including DOM manipulation,
event handling, 
and game state management,
while providing a clean and interactive user experience.

✨ Features

* Interactive two-player gameplay
* Automatic winner detection
* Draw detection
* New Game and Reset Game functionality
* Clean and responsive user interface

 🖥️Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

 📱Getting Started

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.
